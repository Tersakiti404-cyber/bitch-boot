## Install
Follow The Steps Below!

```bash
> termux-setup-storage
(after that tap on permission)
> pkg update -y
> pkg upgrade -y
> pkg install git -y
> git clone https://github.com/Tersakiti404-cyber/bitch-boot
> cd bitch-boot
> bash install.sh
> npm install
> npm audit fix
> npm start
```
